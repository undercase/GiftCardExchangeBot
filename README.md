GiftCardExchangeBot
===================

A bot to monitor /r/giftcardexchange for scammers!

Features:

* Checks Account Age!
* Checks Account Karma!
* Checks if Account is on the Confirmed Scammers List!
* Checks if Account is on the Confirmed Good Traders List!
